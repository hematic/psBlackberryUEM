# Get-AdminRoleUsers

## SYNOPSIS
Function to Get a list of users assigned to the given Admin Role.

## SYNTAX

```
Get-AdminRoleUsers [-guid] <String> [<CommonParameters>]
```

## DESCRIPTION
Function to Get a list of users assigned to the given Admin Role.

## EXAMPLES

### EXAMPLE 1
```
Get-AdminRoleUsers -guid 'f011e00e-88b8-431f-8e15-bed23d266d0c'
```
