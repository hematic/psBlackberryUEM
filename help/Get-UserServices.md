# Get-UserServices

## SYNOPSIS
Function to retrieve services assigned to a user.

## SYNTAX

```
Get-UserServices [-user_guid] <Guid> [<CommonParameters>]
```

## DESCRIPTION
Function to retrieve services assigned to a user.

## EXAMPLES

### EXAMPLE 1
```
Get-BBUEMUserServices -user_guid 'db7c8555-17c2-46b4-be55-86d567f4f7cc'
```

## PARAMETERS
